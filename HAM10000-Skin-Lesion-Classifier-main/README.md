## 🧠 Overview

This project builds an end-to-end skin lesion classification system:

- **EDA** — class imbalance analysis, demographics, pixel statistics
- **Preprocessing** — leak-free splits by `lesion_id`, augmentation, WeightedRandomSampler
- **Baseline CNN** — custom 4-block CNN trained from scratch
- **CNN Model** — EfficientNet-B0 with 2-phase transfer learning (freeze → fine-tune)
- **Evaluation** — confusion matrix, per-class F1, ROC-AUC, Grad-CAM visualizations
- **FastAPI** — REST API serving the trained model with single & batch prediction endpoints

---

## 📂 Dataset

**HAM10000** (*Human Against Machine with 10,000 training images*)  
A large collection of multi-source dermatoscopic images of common pigmented skin lesions.

| Property | Value |
|---|---|
| Total Images | 10,015 |
| Image Size | 450 × 600 px (resized to 224 × 224) |
| Classes | 7 |
| Source | [Kaggle — HAM10000](https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000) |

> Download from Kaggle and place in the following structure:
> ```
> ham10000/
> ├── HAM10000_images_part_1/
> ├── HAM10000_images_part_2/
> └── HAM10000_metadata.csv
> ```

---

## 🏷️ Classes

| Code | Full Name | Type |
|------|-----------|------|
| `nv` | Melanocytic Nevi |  Benign |
| `mel` | Melanoma |  Malignant |
| `bkl` | Benign Keratosis | Benign |
| `bcc` | Basal Cell Carcinoma |  Malignant |
| `akiec` | Actinic Keratosis / Intraepithelial Carcinoma |  Malignant |
| `vasc` | Vascular Lesions |  Benign |
| `df` | Dermatofibroma | Benign |

> ⚠️ The dataset is **heavily imbalanced** — `nv` accounts for ~67% of all images. This is handled via `WeightedRandomSampler` + class-weighted `CrossEntropyLoss`.

## 🔄 Pipeline

```
Raw Images + CSV
      │
      ▼
01_EDA_Preprocessing.ipynb
  ├── Exploratory Data Analysis
  │     ├── Class distribution (imbalance check)
  │     ├── Age / Sex / Localization analysis
  │     ├── Sample image grid per class
  │     └── Pixel channel statistics
  │
  └── Preprocessing
        ├── Missing value imputation
        ├── Label encoding
        ├── Leak-free split by lesion_id (80 / 10 / 10)
        ├── WeightedRandomSampler
        ├── Augmentation (flip, rotate, colorjitter, affine)
        └── Saves → outputs/train.csv, val.csv, test.csv, dataset_meta.json
              │
              ▼
02_Baseline_CNN_Evaluation.ipynb
  ├── Baseline CNN (4-block from scratch)
  ├── EfficientNet-B0
  │     ├── Phase 1 — frozen backbone, train head only
  │     └── Phase 2 — unfreeze all, differential LR fine-tuning
  │
  └── Evaluation
        ├── Classification report (per-class)
        ├── Confusion matrix
        ├── Per-class Precision / Recall / F1
        ├── ROC-AUC curves (one-vs-rest)
        ├── Grad-CAM visualizations
        ├── Misclassification analysis
        └── Saves → models/efficientnet_inference.pth
              │
              ▼
        FastAPI Server
          ├── POST /predict       (single image)
          └── POST /predict/batch (up to 10 images)
```

## ⚙️ Installation

**Python 3.10 or newer recommended**

The original project used PyTorch 2.3 on Python 3.10. If you are using a newer
Python release and see an error like `No matching distribution found for
torch==2.3.0`, install from the updated `requirements.txt`; it allows pip to
choose a compatible PyTorch/torchvision pair.

```bash
# 1. Clone the repository
git clone https://github.com/vanshR18/ham10000-classifier.git
cd ham10000-classifier

# 2. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate        # Linux / Mac
venv\Scripts\activate           # Windows

# 3. Install dependencies
pip install -r requirements.txt
```

## 📓 Running the Notebooks

Open JupyterLab or Jupyter Notebook and run in order:

```bash
jupyter lab
```

### Notebook 1 — EDA & Preprocessing
```
notebooks/01_EDA_Preprocessing.ipynb
```
- Update `BASE_DIR` in **Cell 2** to point to your local `ham10000/` folder
- Run all cells top to bottom
- Outputs saved to `outputs/`

### Notebook 2 — Model Training & Evaluation
```
notebooks/02_Baseline_CNN_Evaluation.ipynb
```
- Automatically loads `outputs/` from Notebook 1
- Trains Baseline CNN (15 epochs) then EfficientNet-B0 (10 + 20 epochs)
- Saves the final model to `models/efficientnet_inference.pth`

> 💡 **GPU strongly recommended.** Training on CPU will be very slow.  
> On an NVIDIA GPU, expect ~30–45 min total training time.

## 🚀 FastAPI Server

Once Notebook 2 is complete and `models/efficientnet_inference.pth` exists:

```bash
# Start the server
uvicorn api.main:app --reload --host 0.0.0.0 --port 8000

The API will be live at:

| URL | Description |
|-----|-------------|
| `http://localhost:8000` | API root |
| `http://localhost:8000/docs` | 📖 Interactive Swagger UI |
| `http://localhost:8000/redoc` | 📄 ReDoc documentation |

|--------|----------|-------------|
| `GET` | `/` | Welcome message + endpoint list |

| `POST` | `/predict/batch` | Batch prediction (up to 10 images) |

## 📤 Example Requests

### Single Image — `curl`
```bash
curl -X POST "http://localhost:8000/predict" \
  -H "accept: application/json" \
  -F "file=@skin_image.jpg"
```

### Single Image — Python
```python
import requests

with open("skin_image.jpg", "rb") as f:
    response = requests.post(
        "http://localhost:8000/predict",
        files={"file": ("skin_image.jpg", f, "image/jpeg")}
    )

data = response.json()
print(f"Predicted  : {data['predicted_class']} — {data['full_name']}")
print(f"Confidence : {data['confidence_pct']}")
print(f"Malignant  : {data['is_malignant']}")
print(f"Warning    : {data['malignant_warning']}")
```

### Batch Prediction — Python
```python
import requests

files = [
    ("files", ("img1.jpg", open("img1.jpg", "rb"), "image/jpeg")),
    ("files", ("img2.jpg", open("img2.jpg", "rb"), "image/jpeg")),
    ("files", ("img3.jpg", open("img3.jpg", "rb"), "image/jpeg")),
]

response = requests.post("http://localhost:8000/predict/batch", files=files)
data = response.json()

print(f"Total images : {data['total_images']}")
print(f"Total time   : {data['total_time_ms']} ms\n")

for result in data["results"]:
    print(f"{result['filename']:<25} → {result['predicted_class']}  ({result['confidence_pct']})")
```
